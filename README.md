# Guía de ejecución de ejercicios Python

Este repositorio contiene los ejercicios prácticos de las lecciones 5, 6 y 7 del curso de Python, centrados en estructuras condicionales, bucles y funciones. A continuación se indican los pasos para ejecutar cada uno de los scripts.

## Requisitos previos

- **Python 3.10 o superior** (para los ejemplos que usan `match-case`, introducido en Python 3.10).  
  Verifica tu versión con:
  ```bash
  python --version
Terminal o consola (PowerShell, CMD, bash, etc.)

  Estructura de archivos

Los scripts se organizan por lecciones y conceptos. Ejemplos de nombres:

    Lección 5 (condicionales): ejemplo_if_edad.py, ejemplo_match_case_fruta.py, etc.

    Lección 6 (bucles): ejemplo_for_range_step.py, ejemplo_while_contador.py, etc.

    Lección 7 (funciones): ejemplo_funcion_saludar.py, ejemplo_return_multiple.py, etc.
